Project 3: Public Works Analysis in Excel and Tableau

In this project, I performed a thorough analysis of public works data in Argentina to gain insights into the distribution and status of infrastructure investments at the regional level, using both Excel and Tableau to optimize data visualization and analysis.

Dataset Source

The dataset was obtained from the Investment Map of Argentina's Ministry of Public Works, which provides detailed information on public works in various provinces and their progress status.

Data Analysis and Preprocessing Techniques

  • Data Cleaning and Validation: Errors were removed and values were validated, both qualitative and quantitative, to ensure the accuracy of the analysis.
  • Pivot Tables: Pivot tables in Excel were used to facilitate data exploration and answer key questions such as the number of works by type, status, and geographic location.
  • Data Segmentation: Segmenters were added to allow dynamic data segmentation by categories such as work type, year, execution status, and province.

Visualization and Dashboards

To enhance data interpretation, interactive dashboards were created in both Excel and Tableau:

  • Excel Dashboard: An interactive dashboard was designed with charts and segmenters to quickly visualize the distribution and status of public works.
  • Tableau Dashboard: An advanced visualization was created to facilitate visual exploration of the data and allow filtering and segmentation of works by various variables of interest.

Key Findings

This analysis provided an overview of the state of public works in Argentina, highlighting regions with the most and least investment, as well as the most common types of projects and their progress status.

Project 4: Popular Festivals in Argentina

This project focuses on the analysis of events and popular festivals in Argentina through a report developed in Power BI. The goal is to explore event attendance, location, and themes.

Dataset Source

The dataset used is a compilation of information on events and popular festivals in Argentina, including details such as the number of attendees, event locations, and themes of the events.

Data Preprocessing Techniques

  • Null Value Control: Handled missing values to ensure dataset integrity.
  • Categorical Variable Encoding: Transformed categorical features for data analysis.

Visualization and Analysis

  • Attendance: Charts showing attendee numbers by month, year, and ticket type.
  • Event Locations: A map shows the distribution of events by province.
  • Theme Analysis: Classifies events by theme and ticket type.

Key Results

This analysis provides a comprehensive view of festival participation in Argentina, highlighting trends in attendance, event distribution, and predominant themes.

Project 5: Spotify Music Data Analysis

This project focuses on analyzing data from Spotify songs to explore musical characteristics and their relation to song popularity.

Dataset Source

The dataset used was downloaded from Kaggle, which provides information on song characteristics such as tempo, valence, energy, and popularity.

Data Preprocessing Techniques

  • Data Extraction: Relevant song data was extracted for analysis.
  • Data Visualization: Visualizations explored the distribution of musical characteristics and their relation to popularity.

Visualization and Analysis

  • Tempo Distribution: A histogram shows tempo distribution among songs.
  • Energy vs. Popularity: A scatter plot illustrates the relationship between song energy and popularity.

Key Results

This analysis provides insights into how musical characteristics influence song popularity on Spotify, revealing trends across different genres.
