Project 2: Weather Prediction Model

This project involves an in-depth analysis of a synthetic dataset designed to classify different climate types.

Data Preprocessing Techniques

  • Null Value Handling and Imputation: Managed missing values to ensure dataset quality.
  • Categorical Variable Encoding: Converted categorical features for model training compatibility.
  • Outlier Removal: Used the interquartile range (IQR) method to eliminate outliers.

Machine Learning Algorithms

  • Random Forest Classifier
  • Gradient Boosting Machine (GBM)
  • Logistic Regression
  • Decision Tree Classifier

Model Optimization and Evaluation

  • Accuracy: Measured the overall correctness of predictions.
  • Confusion Matrix: Assessed classification performance.

Models were compared for optimal weather prediction, and hyperparameter tuning was conducted to improve accuracy.

Project 3: Public Works Analysis in Excel

This project involved an extensive analysis of public works data in Argentina to gain insights into regional infrastructure investments and project status.

Dataset Source

The dataset was sourced from the Investment Map of the Ministry of Public Works of Argentina, detailing public works across provinces and their progress statuses.

Data Analysis and Preprocessing Techniques

  • Data Cleaning and Validation: Removed errors and validated qualitative and quantitative data to ensure accurate analysis.
  • Pivot Tables: Used for dynamic data exploration, facilitating answers to questions about work distribution by type, status, and region.
  • Data Segmentation: Filters were added to allow for real-time segmentation by work type, year, status, and province.

Visualization and Dashboard

An interactive visual dashboard was created in Excel with charts and filters, allowing efficient visualization of work distribution and progress:

  • Distribution of Works by Province: Highlights provinces with the highest infrastructure investment.
  • Classification of Works by Status: Displays work progress (planned, ongoing, completed).
  • Analysis by Work Type: Identifies priority areas in public investment.

Visualization in Tableau

In Tableau, interactive dashboards were created to provide intuitive, real-time visualization of the data, allowing for a dynamic exploration of investments and project progress:

  • Distribution of Works by Province: Visual representation of provinces with the most investment in projects.
  • Classification of Works by Status: Visualization of project progress (planned, ongoing, completed).
  • Analysis by Work Type: Identification of priority categories for public investment.

Key Results

The analysis provided a comprehensive overview of public works status in Argentina, spotlighting regions with the highest and lowest investments and common project types. This insight is crucial for understanding public investment trends and improving work management strategies.
