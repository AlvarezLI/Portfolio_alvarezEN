Project 3: Public Works Analysis in Excel and Tableau

In this project, I performed a thorough analysis of public works data in Argentina to gain insights into the distribution and status of infrastructure investments at the regional level, using both Excel and Tableau to optimize data visualization and analysis.

Dataset Source

The dataset was obtained from the Investment Map of Argentina's Ministry of Public Works, which provides detailed information on public works in various provinces and their progress status.

Data Analysis and Preprocessing Techniques

  • Data Cleaning and Validation: Errors were removed and values were validated, both qualitative and quantitative, to ensure the accuracy of the analysis.
  • Pivot Tables: Pivot tables in Excel were used to facilitate data exploration and answer key questions such as the number of works by type, status, and geographic location.
  • Data Segmentation: Segmenters were added to allow dynamic data segmentation by categories such as work type, year, execution status, and province.

Visualization and Dashboards

To enhance data interpretation, interactive dashboards were created in both Excel and Tableau:

  • Excel Dashboard: An interactive dashboard was designed with charts and segmenters to quickly visualize the distribution and status of public works.
  • Tableau Dashboard: An advanced visualization was created to facilitate visual exploration of the data and allow filtering and segmentation of works by various variables of interest.

Key Findings

This analysis provided an overview of the state of public works in Argentina, highlighting regions with the most and least investment, as well as the most common types of projects and their progress status.

Project 4: Popular Festivals in Argentina

This project focuses on the analysis of events and popular festivals in Argentina through a report developed in Power BI. The goal is to explore event attendance, location, and the theme of each event.

Dataset Source

The dataset used in this project is a compilation of information on events and popular festivals in Argentina, including details such as the number of attendees, event location, and event theme.

Data Analysis and Preprocessing Techniques

  • Null Value Handling: Missing values were properly handled to ensure the dataset's integrity.
  • Categorical Variable Encoding: Categorical features were transformed into numerical values, which is essential for data analysis.

Visualization and Analysis

A Power BI dashboard was developed to explore different aspects of the events:

  • Attendees: Includes graphs showing the number of attendees by month, year, and ticket type.
  • Information: Presents a map displaying the distribution of events by province.
  • Theme: Allows exploration of event themes, showing the classification by ticket type and the number of events by primary theme.

Key Findings

This analysis provides an integrated view of participation in popular festivals in Argentina, highlighting trends in attendance, geographic distribution of events, and predominant themes.

Project 5: Spotify Music Analysis

This project focuses on analyzing Spotify song data with the goal of exploring the musical characteristics and popularity of different tracks.

Dataset Source

The dataset used was downloaded from Kaggle, which provides detailed information about song characteristics such as tempo, valence, energy, and popularity.

Data Analysis and Preprocessing Techniques

  • Data Extraction: A Kaggle dataset was used to extract data about the songs of interest.
  • Data Visualization: Graphs were created to explore the distribution of musical characteristics and their relationship with song popularity.

Visualization and Analysis

Visualizations were developed to explore different aspects of the songs:

  • Tempo Distribution: A histogram was created to show how the tempo of songs is distributed.
  • Energy vs Popularity: A scatter plot was designed to illustrate the relationship between song energy and popularity.

Key Findings

This analysis provides insights into how musical characteristics influence the popularity of songs on Spotify, highlighting trends in different music genres.
